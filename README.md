# Gerador-de-código-com-IA
Um site para gerar códigos com CSS com ajuda da IA

Nesse projeto criamos um site utilizando HTML como estrutura base, CSS para parte visual e Java Script para o operacional.

# Funcionalidades: #
- Geração de linhas de código com IA
- Pré-visualização do código gerado

# Tecnologias Utilizadas: #
- Linguagens: Java Script, HTML e CSS
- IDE: VScode
- IA: llama-3.3-70b-versatile através do Grok

# Pré-requisitos #
Certifique-se de ter as seguintes ferramentas instaladas em sua máquina:
[VScode](https://code.visualstudio.com/download)

# Configuração do ambiente #
1. Clone este repositório

2. Acesse diretório do projeto

3. Abra com o VScode
